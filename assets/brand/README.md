# CoastPulse Logo Direction

## Concept

The mark uses direct coastal imagery:

- Coast: a broad, rounded wave/horizon shape.
- Daily conditions: a small sun accent for beach timing and weather.

This keeps the logo relevant to daily coastal conditions, surf, tides, and beach weather while avoiding graph-like or overly literal data imagery.

## Files

- `coastpulse-logo-bw.svg` - primary structure test and one-color master.
- `coastpulse-logo-color.svg` - primary horizontal color logo.
- `coastpulse-logo-reverse.svg` - dark background version.
- `coastpulse-mark-bw.svg` - one-color app icon, favicon, embroidery, stamp.
- `coastpulse-mark-color.svg` - color app icon and social avatar.

## Practical Notes

- The black-and-white version should be the decision-maker. If the silhouette works there, the color version is support, not a crutch.
- For embroidery or very small print, use the symbol alone and remove the small gold sun if thread/detail limits require it.
- Keep clear space around the logo equal to at least the diameter of the sun.
- Avoid adding outlines, drop shadows, extra wave layers, weather icons, or location-specific text to the core mark.

## Color

- Ocean: `#005F6B`
- Surf: `#1FA5B0`
- Sand/sun accent: `#F2B85C`
- One-color ink: `#111111`

## First-Round Review Checklist

- View it at favicon size and from across the room.
- Flip it horizontally and review for accidental shapes.
- Test it on light, dark, and single-color backgrounds.
- Print it at business-card size before approving the wordmark spacing.
